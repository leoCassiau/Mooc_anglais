Software installation

In this part we will see the main softwares to install on your computer and why these softwares are important. In this tutorial we will only talk about Windows softwares

Let's speak about security

When you buy a new computer the first thing to do is to download and to install an anti-virus. It will protect you against about any kind of malware that you could meet in the web.

One of the most popular anti-virus is avast !

Avast is free and is regularly updated, it's why it's the most commonly used anti-virus. Of course it's also powerful and it can detect and delete most of the malwares present on the web. With a large community of users, if you have a problem it will be easy to solve it.

Web browsers

There are several web browsers, the three most common are Mozilla Firefox, Internet Explorer and Google Chrome

A good web browser is very useful, some of them can't display all web sites properly and their various options. Here we will introduce a good free browser : Mozilla Firefox

Mozilla manages various types of CSS and support HTML5. With this browser you can navigate on any recent web site. Like all the others sofwares there are a large community of user who use this software, so it's pretty easy to solve any kinds of problems you could encounter.

The last version of Internet Explorer can also manage HTML5, but you can have problems with some types of CSS

There is also Google Chrome who is very powerful too

How to keep your computer in shape

CCleaner is a software very useful to keep your computer working properly. This little software will clean your computer in a few clicks. It can delete all your useless temporary files, error files or any files that can hinder your computer

This software can also manage your differents applications (delete, upgrate, repare). Sometimes your computer is very slow to start up, that could have been made by programs that start running when you switch on your computer. CCleaner can protect you from it by deleting the execution of these software

For more information of this product : Doc

As a supplement to CCleaner you also have to think aboug defragmenting your hard disk to improve the speed of your computer (if you possess a SSD, no need for this manipulation). Here is a tutoriel to explain the process : Tuto

Last but not the least : VLC media player

VLC media player is a very good software for music or video player. This software can play any type of video or music format. I have never met a file format it could not read.

Besides being powerful it is free, easy of use, light and has regular updates

When you install Windows, it offers you to use an audio and video player named Windows media. It's a good music player but it can't read all types of video format, so I encourage you to use VLC at least for videos
