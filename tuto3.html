OS Installation

Now, we need to install an Operating System (OS). The OS is the main software of your computer, it will determine the capacities of the PC and the interface you will interact with. It is important to choose an OS that fits yours needs from start, because it is complicated to change your OS.

The most popular OS is Windows. If you are a standard user, you'll choose this OS because it is easy to use (You don't need knowledge in computer science), you have probably already used Windows before and finally, many softwares are only avaible on Windows. The inconvenients are that Windows is not free, it can be resource hungry for a small computer, and this is not an open source software.

If you want to install Windows, you have to choose a release :

  • Windows 10 is the most recent release. Prefer this release to avoid compatibility problems in the future. If you have never used Windows before, this is the most user-friendly version of Windows.
  • Windows 8 is a release avaible on tablet and smartphone, but not otpimized for computers. Avoid this.
  • Windows 7 is a recent release with the main tools of Windows. Today, this is the most popular release and if you have already used an older release, you will not be lost.
  • The olders versions are no longer supported by Microsoft and are now outdated.
You can purchase Windows on a specialized store or on the official website. To install Windows, you just have to put the CD in the DVD player, and follow, step by step, the installation.

Tips : you can use the Windows USB/DVD Download Tool to install your OS by USB.

Ubuntu is a free, a light and open source OS. Ubuntu is the most popular distribution of Linux, and you can download it on the official wesbiste.

This OS is a good alternative to Windows, if you want a small OS. Ubuntu is a user-friendly distribution of Linux, it is really faster than windows, and many softwares are avaible on Linux now. This OS is dedicated to personal computers and to development, but avoid this OS for professional use, because you need an OS similar to the other peoples in your company, and Linux is really different.

To install Ubuntu, you need to download it on the official wesbiste, and to burn it on a CD-ROM. After the download, you will have a single file with the extension .ISO. This extension means this file is an image of a CD, so you need to burn this image on a CD to launch the installation. Many software can burn a CD, we advise you to use IMGBurn , a little CD burner. After that, you can launch Ubuntu with your CD, and follow step by step the installation.

Tips : if your computer already has Windows installed, you can use Wubi to avoid to burn a CD. Wubi will install Ubuntu on your computer like a normal software, and the next time you start up your computer, you will have to choose between Ubuntu or Windows (this is a dualboot).

Other OS

Many other OS exist, but they are less user-friendly. Linux has many distributions, like OS-X, the OS of Mac, or like Debian, an old popular OS, but they are not free or more difficult to use than Ubuntu. So, if you are a novice, we do not recommend to install another OS than Windows or Ubuntu.
